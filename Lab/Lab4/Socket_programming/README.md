
# Socket programming（套接字编程）：这是网络编程的一种技术，允许计算机之间通过网络进行数据交换。一个套接字（socket）是一种通信端点，类似于电话的插孔，它使程序可以发送和接收数据。

# IP address（IP地址）：IP地址是互联网上设备的唯一标识。它用于定位和标识网络上的设备。IP地址有两种主要版本：IPv4（如192.168.0.1）和IPv6（如2001:0db8:85a3:0000:0000:8a2e:0370:7334）。

- 127.0.0.1 -> localhost：这个IP地址被称为“localhost”，表示本地计算机。这是一个特殊的回环地址，用于测试和本地通信。当你访问127.0.0.1时，你实际上是在访问自己的计算机。

- 192.168.xxx.xxx -> local network IP（局域网IP）：这是一个局域网（LAN）中的IP地址。这类IP通常用于家用路由器、局域网内的设备通信，不会在互联网上直接公开。192.168是常见的局域网IP前缀。

- Internet IP address（互联网IP地址）：这是路由器设备在互联网上的唯一地址，用于从互联网上访问设备。这个地址通常由互联网服务提供商（ISP）分配。它相当于你家里的门牌号，用于在庞大的互联网上唯一识别和定位你的设备。在大多数家庭或企业网络中，的确会有两个IP地址。你的家中有一台路由器，它连接到你的互联网服务提供商（ISP），获得了一个公网IP地址，比如 203.0.113.5。这个公网IP就是外界互联网上识别你家网络的唯一地址。你的路由器还会给家里的每台设备分配一个局域网IP，比如你的电脑可能是 192.168.1.10，手机可能是 192.168.1.11，这些都是你的路由器内部的地址。公网IP 允许你的路由器在互联网上与其他设备通信。局域网IP 允许路由器在家里的局域网中管理多台设备，并确保它们能够彼此通信，同时与互联网连接。路由器就像一个“中介”，它通过NAT（网络地址转换）技术，将家中多个局域网IP地址的请求转发到外部的互联网，并使用一个唯一的公网IP来和外部通信。

# Port number（端口号）：端口号用于标识同一设备上的不同服务或应用程序。端口号范围为0~65535。比如，HTTP协议默认使用端口80，HTTPS使用端口443。
